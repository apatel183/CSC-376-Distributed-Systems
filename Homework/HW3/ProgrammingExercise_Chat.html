Programming Exercise

Chat

Due: Wed., Apr. 26, at 10:10am

  • Functionality

    You are to create a chat system. In particular, you will create a chat client and a chat server. You may use your Direct Messenger code as a base for this project, although you will have to revise that code significantly. Notably, the client and the server must be separate programs. Your system will perform as follows:

    1. The server will listen for connections from clients.
    2. Once a client is connected, it can send messages to the server. It will also receive from the server all messages sent from the other connected clients.
    3. Messages must be sent in utf-8 format.
    4. Messages must be provided to the client using standard input.
    5. Immediately after receiving a message, the server must forward the message to all connected clients, except for the client that sent the message. DO NOT SEND THE MESSAGE BACK TO THE ORIGINATING CLIENT!
    6. Once it receives a message, the client will write the message to standard output.
    7. When a client closes its socket, the server will remove that client from its list, but the server will continue to run and the other connected clients will be able to continue exchanging messages.

  • Usage

    • Follow the same usage as described for the Direct Messenger.
